Clone Groceries - Pass by Value vs Pass by Reference

Key Concepts

Pass by Value (Primitives): string, number, boolean, null, undefined, symbol
When you copy a primitive value, you create a completely independent copy.

Pass by Reference (Objects): object, array, function
When you copy an object, you copy the reference (memory address), not the actual object.
Both variables point to the same object in memory!
